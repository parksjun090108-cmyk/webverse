# WebVerse Visit Sync

## 로컬 설치

1. WebVerse를 실행합니다.
2. Chrome/Edge 주소창에서 `chrome://extensions`를 엽니다.
3. 개발자 모드를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다**를 누릅니다.
5. 이 프로젝트의 `extension` 폴더를 선택합니다.
6. WebVerse의 설정 → 브라우저 자동 동기화에서 연결 코드를 만듭니다.
7. 확장 프로그램 팝업에 코드를 입력합니다.
8. 최근 30일 방문 상위 20개와 북마크 목록에서 제외할 사이트의 체크를 해제합니다.
9. **선택한 사이트로 우주 만들기**를 누릅니다.

확장 프로그램은 계정 연결 전에는 방문 정보를 수집하지 않습니다. 첫 우주 목록은 브라우저 안에서 계산되며 체크한 사이트만 서버로 전송합니다. 연결 후에도 방문한 전체 URL을 메모리에 오래 보관하지 않고 즉시 도메인만 추출합니다. 동일 도메인의 30초 이내 반복 방문은 한 번으로 처리하며, 네트워크 오류 시 최대 500개의 방문·북마크 이벤트를 브라우저 내부 저장소에 대기시킵니다.

브라우저 북마크를 추가하거나 삭제하면 WebVerse 즐겨찾기 고리에 단방향으로 반영됩니다. WebVerse에서 즐겨찾기를 변경해도 브라우저 북마크를 생성하거나 삭제하지 않습니다.

운영 배포 전 `manifest.json`의 `host_permissions`에 실제 Render API 주소를 추가하고 `popup.html` 및 `background.js`의 기본 API 주소를 운영 주소로 변경해야 합니다.
